# kore-lang
*What is Kore?*

Kore is an interpreted language focused on convinience and redability.
Code is easily written when the language its written in is focused on redability and making the developers job easier, kore does this by having a simple, english-like syntax, and features focused on the experience of anyone using it.

The name "Kore" comes from merging the words convenient and redable, taking the first 2 letters from each word, forming "core" though a stylistic choice was made changing the c to a k since it looks cooler(Kore>Core)
Kore aims to help out scripters and beginners in the world of coding write scripts and simple projects, for I beilive that a simple projects written for fun shouldn't have its creator awake at night with ever increasing bugs or be difficult to mantain.

*Installation*

Kore does not have a way of being installed nor used, *yet*

*Usage*

Kore aims to be easy to use, hence theres multiple pre-built functions ready to run, some of which include the following:
print()
is_prime()
reverse()
sort()
split()
replace()
XOR()

For the documentation of the language please refer to the DOCS.md file on the github repository.
